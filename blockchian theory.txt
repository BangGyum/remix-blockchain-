스마트 계약 또는 스마트 컨트랙트(smart contract)는 계약 당사자가 사전에 협의한 내용을 미리 프로그래밍하여 
전자 계약서 문서 안에 넣어두고, 이 계약 조건이 모두 충족되면 자동으로 계약 내용이 실행되도록 하는 시스템이다. 

스마트계약은 중개인이 필요없음
스마트계약이 블록체인 상단에서 작동한다는 것 = 문서가 공유원장에 암호화된다는 의미
스마트계약은 항상 백업되기에 손실 걱정이 없음.
스마트계약은 암호화되어  안전이 확보됨 (코드 조작되는 일은 매우 드뭄.)
기존 계약 프로세스에 비해 속도가 매우 빠름. (법률 계약에서 시간을 많이 절약)
그리고 절약됨. (속도가 빠르니깐)
그리고 정확함. (기존 계약은 많은 사람이 관여하지만, 소프트웨어가 보장)

-------112 '계정' 과 ' 계약 지갑' 간의 차이
-계정
 코드로 제어되지 않음. 반면, 계약 지갑은 코드에 의해 제어됨.

 스마트 지갑의 유일한 목적 = 이더를 받고, 저장하고, 보내는 것
  때문에 계정에 비하면 훨씬 편하고 설정 변경도 쉬움
Contract Wallets are controlled By Code

코드는 마스터를 가지고 있음 ( 마스터 = 계정 )
-계정은 두가지 역할 : 이더 보관, 스마트 지갑의 소유자
-게정은 모든 거래를 나열 할 수 없다. (스마트 지갑은 가능)

- 스마트 게약은 실제 다중서명 지갑으로 설정 가능. 
    (다중서명지갑은 거래를 위해 추가적인 보완 요구하게 됨)
    기본적으로 블록체인에 전달되어 노드에 의해 입증되기 전, 거래 서명을 위해 다른 사용자 or 다른 확인 과정을 요구
        이 과정은 가스비용이 발생 ( 계정은 아무런 비용 발생안함)
스마트 지갑으로 다양한 시도를 할 수 있도록 '계정에 이더를 보관하는 것을 항상 권장'
